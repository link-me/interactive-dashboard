# Interactive Dashboard — Problem & Value Overview

## Problem
- Decision latency: KPI разбросаны, наглядности мало, сложно увидеть тренды быстро.
- Low signal-to-noise: сырой табличный вид не подсвечивает сезонность и изменения.
- Alignment: у стейкхолдеров нет общего “одного экрана” для обсуждения метрик.

## Value Proposition
- Single-screen trends: график с 7‑дневным средним показывает динамику без шума.
- Instant conversion insight: сводка считает итоговые значения и коэффициент конверсии.
- Zero‑setup demo: работает из браузера — открыть `src/index.html` и сразу смотреть.

## Use Cases
- Product/Analytics: быстрый обзор Visits, Signups, Conversions за 3/6/12 месяцев.
- Marketing: оценка влияния кампаний, акций и контент‑планов на воронку.
- Engineering: прототип интеграции метрик (замена генератора на реальный API).

## Non‑Goals (сейчас)
- Хранение и аутентификация пользователей.
- BI‑функции: сегментация, дашборды с большим числом источников, экспорты.
- Продакшн SLA: это демо без серверной части и ролей.

## Roadmap
- Подключение реальных источников данных (REST/GraphQL), кэширование, веб‑сокеты.
- Аннотации событий на графике (релизы, кампании), сравнение периодов (YoY, WoW).
- Экспорт (CSV/PNG), пресеты метрик, сохранение фильтров в `localStorage`.

## Metrics
- Visits, Signups, Conversions.
- Conversion Rate = Conversions / Signups.
- Скользящее среднее (7д) для сглаживания шумов.

## Accessibility & Performance
- Легенда и тултипы читаемы, контраст тёмной темы.
- Нет сборки: лёгкие статические файлы + CDN Chart.js.
